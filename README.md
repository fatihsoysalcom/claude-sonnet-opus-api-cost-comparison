# Claude Sonnet Opus API Cost Comparison

This example demonstrates the significant cost difference between Anthropic's Claude 3 Opus and Claude 3.5 Sonnet models based on their official API pricing. It simulates various API call scenarios with different input and output token counts, calculating and comparing the estimated costs for each model. The script highlights how Sonnet offers substantial savings, aligning with the article's discussion on AI budget optimization.

## Language

`python`

## How to Run

1. Ensure you have Python installed (version 3.6 or higher).
2. Save the code as `main.py`.
3. Run from your terminal: `python main.py`
